# Fantasy F1 League

Live standings, driver points, and race history for your Fantasy F1 league.

## Making Updates

To update rosters during the season, edit `src/App.jsx` and find the `INITIAL_ROSTERS` section near the top of the file. Change the driver codes there and commit — Netlify will auto-redeploy.

## Local Development (optional)

```
npm install
npm start
```
